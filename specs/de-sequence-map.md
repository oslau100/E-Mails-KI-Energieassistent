# DE Sequence Map

Hinweis: Jede E-Mail enthält genau einen CTA-Link (offer oder referral) plus den Absenderkontakt im Footer.

## Qualitätsregeln
- Betreffzeile: konkret, persönlich, nicht irreführend
- Preheader: ergänzt den Betreff ohne Fake-Dringlichkeit
- Doppeldeutige Begriffe vermeiden
- {{ opportunity.energieart }} selektiv nutzen
