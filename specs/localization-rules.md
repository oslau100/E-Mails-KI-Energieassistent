# Localization Rules

## Current state
Phase 1 = German master emails.

## Keep across languages
- same funnel logic
- same scenario psychology
- same plain personal style
- same CTA intent

## Subject + preheader portability
- concrete, clear, non-spammy
- no fake urgency
- no fake support/transaction framing
- preserve selective personalization intent (incl. energy type variable)

## Variable handling
- preserve placeholders exactly
- {{ opportunity.energieart }} can be used selectively for relevance
- do not invent fields

## Bridge-line rule
When translating/adapting, preserve selective single-line objection/trust bridges where used.

## Forbidden ambiguous wording (or equivalents)
- „dein aktueller Stand"
- „dein Tarif-Status"
- „deine Situation"
- „deine Daten"

## Signature consistency
Preserve compact signature block structure (brand, logo, address, impressum, datenschutz) across localized variants.
